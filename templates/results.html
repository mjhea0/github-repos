What is this?

Your Github Repositories.

Why did I make this?

One way that I learn is by looking at how other people solve problems. I created this for myself, but thought that others may learn in the same way as I do. Sharing is caring.

How did I do this?

In short, I consumed the Github API and parsed the data. It's built using Python using a web framework called Flask. If you want to know more, just ask.

How is the data organized?

If your repo has the name "refactor" (case insensitive) in it, you should see it in the list. I used a quick and dirty method of retrieving the data, as well as parsing and organizing it. My goal is to use a more sophisticated means of organizing the data and delivering it in a more logical manner, unless Matt "Empire" Null builds out this functionality into the student portal.
